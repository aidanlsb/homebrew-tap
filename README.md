# homebrew-tap

Homebrew tap for [Raven](https://github.com/aidanlsb/raven) (formula: `rvn`).

## Install

```bash
brew tap aidanlsb/tap
brew install aidanlsb/tap/rvn
```

## Upgrade

```bash
brew update
brew upgrade aidanlsb/tap/rvn
```

## Publishing

Formula updates are pushed by GoReleaser from `aidanlsb/raven` to this repository.
